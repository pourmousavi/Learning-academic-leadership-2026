# HOW TO USE YOUR ACADEMIC LEADERSHIP APP

## Quick Start Guide

You now have a fully functional Progressive Web App (PWA) for your Academic Leadership 2026 program! This app works on your iPhone (or any device) and can be installed to your home screen like a native app.

---

## INSTALLATION OPTIONS

### Option 1: Host it Online (Recommended for iPhone)

**Best for:** Easy access from anywhere, automatic sync across devices

**Steps:**
1. Upload these files to a web hosting service:
   - index.html
   - app.js
   - manifest.json
   - sw.js
   - icon-192.png
   - icon-512.png

**Free hosting options:**
- **GitHub Pages** (recommended): 
  - Create a GitHub account
  - Create a new repository
  - Upload all files
  - Enable GitHub Pages in settings
  - Access at: https://[your-username].github.io/[repo-name]

- **Netlify** (easiest):
  - Go to netlify.com
  - Drag and drop all files
  - Get instant URL
  - Access from anywhere

- **Vercel**:
  - Similar to Netlify
  - Free tier available
  - Very fast

### Option 2: Access Locally

**For testing on your computer:**
1. Open Terminal
2. Navigate to the folder with the files:
   ```
   cd /path/to/your/files
   ```
3. Start a simple server:
   ```
   python3 -m http.server 8000
   ```
4. Open browser and go to: http://localhost:8000
5. Access from phone on same WiFi: http://[your-computer-ip]:8000

## HOW TO INSTALL ON IPHONE

Once you have the app hosted online:

### Installation Steps:

1. **Open Safari** (must use Safari on iPhone)
2. **Navigate to your app URL**
3. **Tap the Share button** (square with arrow pointing up)
4. **Scroll down** and tap "Add to Home Screen"
5. **Tap "Add"** in the top right
6. **Done!** The app now appears on your home screen

### What You Get:
✓ App icon on home screen
✓ Full-screen experience (no browser UI)
✓ Works offline after first load
✓ Fast access like a native app
✓ Progress saved locally on your device

## TROUBLESHOOTING

### App won't install on iPhone?
- Must use Safari (not Chrome)
- Make sure you're on the actual website (not localhost)
- Try clearing Safari cache

### Progress not saving?
- Check if browser has enough storage
- Don't use Private/Incognito mode
- Export progress data as backup

### Content not loading?
- Check internet connection on first load
- Refresh the page
- Clear browser cache

### Want to sync across devices?
- Currently saves locally per device
- Export progress data from one device
- Can manually import on another (future feature)

### Reset progress accidentally?
- If you backed up, you can restore
- Otherwise, start fresh
- This is why weekly exports are recommended

## DATA AND PRIVACY

**Where is my data stored?**
- Locally on your device only
- Uses browser localStorage
- Never sent to any server
- Completely private

**What if I lose my phone?**
- Data is gone unless you exported it
- Recommendation: Export monthly
- Save exports to cloud storage

**Can others see my progress?**
- No, it's device-specific
- Each device has separate progress
- Perfect for privacy

## EXAMPLE SETUP WITH GITHUB PAGES

**Step-by-step for complete beginners:**

1. Go to github.com and create free account
2. Click "New repository"
3. Name it: "academic-leadership-2026"
4. Make it Public
5. Click "Create repository"
6. Click "uploading an existing file"
7. Drag all 6 files into the box:
   - index.html
   - app.js
   - manifest.json
   - sw.js
   - icon-192.png
   - icon-512.png
8. Click "Commit changes"
9. Go to Settings → Pages
10. Under "Source" select "main branch"
11. Click Save
12. Wait 2 minutes
13. Your URL will be: https://[your-username].github.io/academic-leadership-2026
14. Open this URL in Safari on your iPhone
15. Add to Home Screen
16. Done!
